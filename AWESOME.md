## Submission rules

To keep this list useful, each entry must:

1. Still work with the current Claude Code version
2. Have a one-line reason why a newcomer should care
3. Not be a personal blog post that duplicates the official docs
4. Not be outdated within 12 months (we prune regularly)

Old tip lists and video-of-the-month write-ups belong in a Discussion, not here.
